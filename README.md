# ep_trackersystem
a simple but cool tracker system. Works with ESX, QBX & QBCore. 

## Showcase 
- https://cap.link/sndk83nkhrxx62r

## Requirements

- [ox_lib](https://github.com/overextended/ox_lib)

## Installation

1. Add the resource as `ep_trackersystem`.

2. `ensure ep_trackersystem` after dependencies.


## Configuration

| File | What to edit |
|------|----------------|
| `locales/` | Add lanuages or edit one |
| `config.lua` | Select the locale that you want to use |
